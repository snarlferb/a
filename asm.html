Data Section:   - Purpose: Reserved for initialized data, constants, and possibly space for uninitialized data (BSS).     . Text Section (Code Section):   - Purpose: Contains the executable code.     . Function Prologue:   - Purpose: Prepares the function for execution.   - Common Steps:   - Save register values that need to be preserved (push).   - Allocate space for local variables and function parameters.     . Function Epilogue:   - Purpose: Cleans up after the function execution.   - Common Steps:   - Deallocate space for local variables and parameters (add esp).   - Restore saved register values (pop).   - Return from the function (ret).     .
